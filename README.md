# Food segmentation for counting calories
## Requirements
- Tested on `python3.7`  
- CUDA  

Run to install libraries:   
```bash
chmod +x install.sh  
./install.sh
```  
## Usage  
```
python train.py  
python test.py
```
Run following to get information about arguments:  
```
python train.py -h
python test.py -h
```  
Change configs in `config.py` to fine-tune your model.
